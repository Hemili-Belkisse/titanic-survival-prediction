
Titanic Survival Prediction

## What it does
Predicts whether a Titanic passenger survived or died
based on their age, gender, ticket class and fare.

## Dataset
Kaggle Titanic Competition
kaggle.com/competitions/titanic

## Models used
- Logistic Regression (Scikit-learn)
- Decision Tree Classifier (Scikit-learn)

## Results
Logistic Regression accuracy : 77.1%
Decision Tree accuracy       : 77.1%
Kaggle submission score      : 0.76
